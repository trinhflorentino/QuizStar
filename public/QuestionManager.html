Nhập đề bằng file Excel

Nhập đề nhanh bằng file Excel, tải xuống, chỉnh sửa và thêm vào bộ đề của bạn.

Hướng dẫn: Chuẩn bị trước file Excel mẫu không chứa dữ liệu, sau khi chọn Xuất file Excel, chọn file mẫu đã tải xuống, tiếp theo mới chọn chọn đường dẫn xuất file Excel.
Lưu ý: File xuất sẽ không giữ được style mẫu ban đầu của file gốc, nhưng vẫn có thể dùng file đã xuất để import lại được bình thường.

Số lượng thí sinh

Chọn số lượng thí sinh tham gia trận đấu

Lưu ý: Thay đổi này sẽ ảnh hưởng đến toàn bộ hệ thống và các màn hình khác. Vui lòng thay đổi trước khi cung cấp mã trận và mật khẩu cho thí sinh. Reload lại trang Quản lý câu hỏi, trang Điều khiển bằng F5 để cập nhật giao diện. Sau khi chọn số lượng thí sinh, chọn Lưu thay đổi để áp dụng.
Tác động: Số lượng gói câu hỏi trong Khởi động I và Về đích sẽ thay đổi theo số lượng thí sinh được chọn.
